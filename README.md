# Slides

This repository hosts the slides I build for teaching / talks. I usually build my slides using HTML technologies such as the [{xaringan}](https://github.com/yihui/xaringan) R :package: (along with some institutional CSS I made).

Each sub-directory contains the slide source code and a README giving the link to the presentation deployed using Github Pages.

If you're interested in the CSS template, see the [{InraeThemes}](https://github.com/davidcarayon/InraeThemes) R :package: .
